# Markdown Cheatsheet

Quick reference for common markdown syntax.

## Text Formatting

- **Bold** with `**text**`
- *Italic* with `*text*`
- `Code` with backticks
- ~~Strikethrough~~ with `~~text~~`

## Lists

Ordered:
1. First
2. Second
3. Third

Unordered:
- Item A
- Item B
- Item C

## Links and Images

```md
[Link text](https://example.com)
![Alt text](./image.png)
```

## Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
