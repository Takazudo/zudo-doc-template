# How to Add a Page

Create a new `.md` file in the appropriate category directory.

## Steps

1. Create a markdown file (e.g., `my-page.md`)
2. Add frontmatter with `title` and `sidebar_position`
3. Write your content
4. The page appears in the sidebar automatically

```md
---
title: My New Page
sidebar_position: 5
---

# My New Page

Content goes here.
```
